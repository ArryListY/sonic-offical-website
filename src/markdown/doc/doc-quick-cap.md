# 快速截图

这里是快速截图的使用教学。

## 马上截图

> 1. 点击截图。
> 2. 完成！当前投屏画面的截图已经截取下来了。
> 3. 接下来可以点击保存，会直接下载到本地。

当截图过多时，也可以点击【清空】，清空当前已截的图片。

## 关于iOS快速截图

> iOS设备的快速截图功能在2.0.2版本开始可以使用，使用体验与安卓无异。
