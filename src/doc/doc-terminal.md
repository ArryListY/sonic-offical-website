# Terminal

这里是Terminal的使用方法介绍。

## 安卓

### Shell

同 adb shell，在使用时不需要输入adb shell，直接输入对应指令即可。例如：
> ```
> wm size
> ```

### Logcat
> 同adb logcat，使用时可以筛选logcat日志等级与关键字。

## iOS

### Process

> 同 sib ps，展示当前实时运行进程列表。

### SysLog

> 同 sib syslog，展示当前系统日志。
