# 运行测试

这里是远控时进行UI自动化的教学。

## 开始运行

运行前先确保Driver已初始化完毕。
> 1. 选择对应项目
> 2. 选择对应测试用例
> 3. 点击【开始运行】

## 运行结果

可以直接点击测试报告查看当前运行结果。
