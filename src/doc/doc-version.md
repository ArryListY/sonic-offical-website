# 版本管理

对项目进行版本管理。

## 添加版本迭代

1. 前往首页，点击目标项目。
2. 点击【项目相关设置】选项。
3. 点击【版本迭代】选项。
4. 点击【添加版本迭代】按钮。
5. 输入版本名称和时间。
6. 点击【确定】按钮。
7. 版本迭代添加完毕！

## 编辑版本迭代

1. 前往首页，点击目标项目。
2. 点击【项目相关设置】选项。
3. 点击【版本迭代】选项。
4. 点击目标版本迭代的【编辑】按钮。
5. 编辑你需要更改的信息。
6. 点击【确定】按钮。
7.  版本迭代编辑完毕！

## 删除版本迭代

1. 前往首页，点击目标项目。
2. 点击【项目相关设置】选项。
3. 点击【版本迭代】选项。
4. 点击目标版本迭代的【删除】按钮。
5. 再次点击【确定】按钮。
6. 版本迭代删除完毕！
